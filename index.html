## When Use Entityform? When you want to: * Use standard field types * Integrate with Views, Rules, other Drupal modules * Export easily to Features * Implement complex notifications
## When Use Webform? When you want to: * Have lots of different forms * Give a sandboxed interface to end users * Create forms with LOTS of fields (150+) * Use built-in analysis tools
